Citas

HTML también tiene etiquetas que ayudan a darle un formato rápido a las citas textuales tanto cortas como largas.

<q>

Normalmente el texto que se encuentra entre <q> y </q> es deplegado como una cita corta; es decir, solamente se le añade comillas dobles al inicio y al final de la cita.

<blockquote>

Esta etiqueta da el formato a una cita extensa: identación de ambos lados.

<abbr>

La etiqueta <abbr> sirve para dar el significado de las abreviaciones, traducciones o conceptos al usuario.

<address>

La etiqueta <address> añade formato al texto en el cual se encuentra la información de contacto del autor.

<cite>

La etiqueta <cite> se usa para escribir el nombre o título de alguna obra. Comúmnente esta etiqueta se despliega como si fueran letras itálicas.

<bdo>

Esta etiqueta voltea.
