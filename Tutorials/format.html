Formato de texto

De manera similar al atributo style, existen elementos que permiten dar formato al texto en cualquier momento.

<b> y <strong>

<b> es una etiqueta que se utiliza para poner en negritas el texto que esté entre <b> y </b>; aunque <strong> parece que hace lo mismo, se utiliza para darle una importancia extra al texto.

<i> y <em>

En caso similar, la etiqueta <i> le da formato al texto en letras cursivas y <em>, además de hacerlo cursivo, le da importancia al texto.

<small>

La etiqueta <small> hace más pequeño al texto que se escriba entre <small> y </small>.

<mark>

La etiqueta <mark> marca al texto que se escriba entre <mark> y </mark> como si se estuviera usando un marcatextos.

<del>

La etiqueta <del> tacha el texto que se escriba entre <del> y </del> para simular que el texto fue retirado.

<ins>

La etiqueta <ins> subraya el texto que se escriba entre <ins> y </ins> para simular que el texto fue añadido o insertado.

<sub>

La etiqueta <sub> escribe el texto entre <sub> y </sub> como un subíndice.

<sup>

La etiqueta <sup> escribe el texto entre <sup> y </sup> como un superíndice.



¡Importante!
  • Aunque el explorador despliegue las etiquetas <b>, <strong>, <i> y <em> de manera similar, las etiquetas <strong> y <em> quieren decir que el texto es importante.
