Headings (< h1 >)

Los headings son las etiquetas que se usan para los encabezados y partes importantes del documento HTML. Los encabezados van desde < h1 > que define el más importante, < h2 >, ..., hasta < h6 >, el de menor importancia. El texto del encabezado se escribe entre las etiquetas de apertura y de cierre: < h1 > Encabezado 1< /h1 >.

Los encabezados son importantes ya que ayudan a las herramientas de búsqueda a encontrar la estructura y contenido de la página web.

Se puede utilizar la etiqueta < hr > para separar el contenido o definir un cambio en la página html. Esta etiqueta se despliega como una línea horizontal, ideal para colocarse entre los encabezados y sus respectivas secciones


Párrafos (< p >)

El texto que se visualiza en una página web se escribe entre las etiquetas < p > y < /p >. Es importante tomar recordar que HTML no toma en cuenta la manera en que escribamos el texto, ya que el formato como lo despliegue el explorador va a ser diferente y sin "espacios extras". Por esta razón podemos incluir espacios entre líneas con < br > (esta etiqueta no requiere una etiqueta de cierre).

Imágenes (< img >)

Al insertar una imagen solamente se usa la etiqueta de abertura: < img >. Como atributos se especifica src donde va el url de una imágen de internet o el path de un archivo local; de manera opcional se especifican heigth y width para determinar el tamaño de la imagen; y también el atributo alt que añade un texto en caso de no encontrar la imagen especificada.
