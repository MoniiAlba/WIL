Atributos

Los atributos son los que permiten agregar información importante de la presentación de cierto elemento html. Estos se especifican en la etiqueta de apertura y normalmente se presentan de la siguiente manera: <etiqueta nombreAtributo = "valorAtributo">.

style

Se puede utilizar tanto en el documento html en general como en elementos de este en cualquier momento. Sirve para especificar color, fuente de letra, tamaño de letra, posición, color de fondo, etc. En esta sección el valor del atributo es una propiedad de CSS (el cual se verá más adelante).

lang

Sirve para definir el idioma del documento html, lo que puede servir para algunas aplicaciones y herramientas: <html lang = "es">

title

Este atributo se añade a las etiquetas para que se despliegue el tírulo del párrafo cuando el cursor se posa sobre el elemento señalado.



¡Importante!
  • Aunque al escribir los atributos de las etiquetas, las mayúsculas o minúsculas no influyan es preferible escribirlos con minúsculas.
  • Al escribir los valores de los atributos son necesarias las comillas, ya que de otra forma pueden dar resultados incompletos o no deseados.
  • Si se requiere escribir un texto entre comillas dentro de un atributo se puede solucionar de las siguientes maneras: <etiqueta nombreAtributo = "valor 'del' atributo"> o <etiqueta nombreAtributo = 'valor "del" Atributo'>, dependiendo de cuáles comillas se quieran ver reflejadas en el texto.
