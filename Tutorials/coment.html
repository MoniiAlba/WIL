Comentarios

Los comentarios son elementos muy útiles cuando estamos programando en cualquier tipo de lenguaje. Sirven para escribir cosas sin que afecte ni se despliegue en el resultado final. Son utilizados como guía y para fácil mantenimiento (por parte del autor o alguien más) del código o de la página web. Los comentarios se escriben entre "<!-- Aquí va un comentario -->"

<abbr>

La etiqueta <abbr> sirve para dar el significado de las abreviaciones, traducciones o conceptos al usuario.

<address>

La etiqueta <address> añade formato al texto en el cual se encuentra la información de contacto del autor.

<cite>

La etiqueta <cite> se usa para escribir el nombre o título de alguna obra. Comúmnente esta etiqueta se despliega como si fueran letras itálicas.
